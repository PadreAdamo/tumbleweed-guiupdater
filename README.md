# tumbleweed-guiupdater
KDE-native system update orchestrator for openSUSE Tumbleweed.

Tumbleweed GUIUpdater (name pending) is a KDE-native **system update orchestrator** for openSUSE Tumbleweed.

It provides a simple, policy-driven way to keep a Tumbleweed system up to date using
the recommended `zypper dup` workflow — without exposing package management complexity.

## What this is
- A one-button system updater
- Safe defaults for a rolling release
- Optional automation, notifications, and Snapper integration

## What this is not
- A package manager
- A replacement for zypper, YaST, or Myrlyn
- An application store
